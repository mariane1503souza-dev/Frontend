Aqui está a explicação detalhada de cada uma dessas tags HTML que você usará no seu exercício:

| Tag | Nome | O que ela faz? |
| :--- | :--- | :--- |
| **`<h1>`** | Título Principal | Define o título mais importante da página. Deve haver apenas um por página para ajudar na organização e no SEO (buscas do Google). |
| **`<h2>`** | Subtítulo | Usada para títulos de seções dentro da página. É um nível abaixo do `<h1>`. |
| **`<p>`** | Parágrafo | Usada para blocos de texto comum. O navegador adiciona automaticamente um pequeno espaço antes e depois de cada parágrafo. |
| **`<ul>`** | Lista Não Ordenada | Cria uma lista de itens onde a ordem não importa (geralmente exibe "bolinhas" ou *bullets*). |
| **`<ol>`** | Lista Ordenada | Cria uma lista numerada (1, 2, 3...) onde a sequência dos itens é importante. |
| **`<li>`** | Item da Lista | É a tag que vai **dentro** de `<ul>` ou `<ol>` para representar cada item individual da lista. |
